  • Sept 2020

    License Plate Recognition

    A plate recognition system that understands license plate. Using the pre-trained Wpod-Net model to detect and extract license plates of vehicles images. Using the pre-trained MobileNet model to predict the characters in the extracted image.

    Extracting license plate image, and then convert to grayscale and blur the image, apply inversed thresh_binary, apply dilation, and apply contours as the preprocessing techniques. Then, predict the characters on the image using MobileNet.

  • Sept 2020

    Speech Recognition

    A speech recognition system that understands simple spoken commands.

    With 30 commands, and ~60,000 files. Implementing model using conv1d, and then train model on a batch size of 32 with 100 epoch. End up with 89% train accuracy, and 83% test accuracy. Saving the best model after every epoch as the validation data for prediction.
